# 基于BL0910的10通道电能表
10 channel power meter by BL0910 with esphome
## 基本信息
依据贝岭BL0910芯片设计；</br>
ESP8266使用ESP-01S；</br>
默认使用UART通讯，TX/RX需要上拉电阻；</br>
预留SEL、CS、SCLK引脚，通过对应的电阻进行选择，可改变UART通讯速率/使用SPI通讯；
